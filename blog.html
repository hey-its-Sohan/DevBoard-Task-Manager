Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to select elements in DOM. For example, we can access elements by getElementByID(“id”), getElementsByClass(“class”), getElementsByTagName(“tag”), querySelectorAll("selector") etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML lets us change the content in an element. For example, we can change texts in bold format or give some other html commands. Otherside, innerText gives access to the texts inside an element. It will let you change or add or remove text inside an element. innerText only accesses the visible text but textContent selects all the text even if it’s hidden.

Question-3: What is event delegation in the DOM?

Event delegation means adding an event listener to a parent element instead of each child element. When a child is clicked, the event moves up to the parent, which handles it.

Question-4: What is event bubbling in the DOM?

Event bubbling happens when an event starts on the element is clicked and then moves up to its parent elements. For example, if we click a button inside a "div", the button handles the event first, then the "div", and then the page.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, we can create an element using document.createElement("tagName") then add it using appendChild("createdElement"), and remove it using remove() or removeChild().
